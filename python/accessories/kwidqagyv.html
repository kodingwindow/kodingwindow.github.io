Step 1: Open a terminal -> login with super/root user and execute the following commands
The objective of this segment is to install, setup, and create a project in Django. So, let’s get started with a few steps to start and run the first project.
Visit the official website of Django project and check the latest version of Django, and then use the following steps to install it.

Step 2: Create the first project
kodingwindow@kw:~$ python3 -m django --version
3.1

django-admin startproject YOUR_PROJECT_NAME
kodingwindow@kw:~$ django-admin startproject BankPortal
