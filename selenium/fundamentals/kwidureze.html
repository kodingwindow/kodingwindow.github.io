Selenium Python to launch the Firefox browser and navigate to a given URL
{{ site.pyfile }}
{% highlight python %}
from selenium import webdriver

driver = webdriver.Firefox(executable_path='D:\Selenium\Drivers\geckodriver.exe')
driver.maximize_window()
driver.get("https://kodingwindow.com/testapp/")
{% endhighlight %}
