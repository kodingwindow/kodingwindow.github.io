mv command to rename a file
godarda@gd:~$ ls
a.out  gd.cpp   gd.py    Documents  Fedora.txt  Pictures  Templates   Videos
gd.c   GD.java  Desktop  Downloads  Music   Public    Ubuntu.txt  Weekdays

godarda@gd:~$ mv Fedora.txt FEDORA.TXT
godarda@gd:~$ ls
a.out  gd.cpp   gd.py    Documents  FEDORA.TXT  Pictures  Templates   Videos
gd.c   GD.java  Desktop  Downloads  Music   Public    Ubuntu.txt  Weekdays
cp command to copy a file
godarda@gd:~$ cp FEDORA.TXT /home/godarda/Documents/
godarda@gd:~$ ls $cat /home/godarda/Documents/
FEDORA.TXT
cp and mv commands to copy and move the files
godarda@gd:~$ mkdir Linux
godarda@gd:~$ touch Ubuntu.txt Fedora.txt
godarda@gd:~$ cp Ubuntu.txt Fedora.txt
godarda@gd:~$ ls
a.out   GD.java  Documents   Linux     Public      Videos
gd.c    gd.py    Downloads   Music     Templates   Weekdays
gd.cpp  Desktop  Fedora.txt  Pictures  Ubuntu.txt

godarda@gd:~$ cp Ubuntu.txt /home/godarda/Linux/
godarda@gd:~$ ls $cat /home/godarda/Linux/
Ubuntu.txt

godarda@gd:~$ mv Fedora.txt /home/godarda/Linux/
godarda@gd:~$ ls $cat /home/godarda/Linux/
Fedora.txt  Ubuntu.txt

godarda@gd:~$ ls
a.out  gd.cpp   gd.py    Documents  Linux  Pictures  Templates   Videos
gd.c   GD.java  Desktop  Downloads  Music  Public    Ubuntu.txt  Weekdays
godarda@gd:~$ cd Linux/
godarda@gd:~/Linux$ ls
Fedora.txt  Ubuntu.txt
