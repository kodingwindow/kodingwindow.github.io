Steps to download MongoDB

1. Go to MongoDB download center
2. Click MongoDB Community Server tab
3. Select OS as Ubuntu
4. Select Package as .tzg
5. Click download button

Step-1: Extract .tgz into any directory (right click -> select option Extract Here)
Step-2: Create folders data/db in the extracted path
Step-3: Open a terminal and execute the following steps
Note: don’t close this instance of mongod terminal, until and unless you finish you tasks.
